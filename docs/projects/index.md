    <ul>
      <li>支持单路摄像头、图片和短视频媒体接入</li>
      <li>支持已知人脸保持清晰、陌生人脸自动模糊</li>
      <li>支持人脸库管理、识别阈值和模糊强度调节</li>
      <li>支持处理后媒体导出和基础运行日志记录</li>
